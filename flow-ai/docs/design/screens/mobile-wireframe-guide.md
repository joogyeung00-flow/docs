## 1. 앱 연결 케이스

### 플로우 → 플로우 AI 진입 경로
1. **대시보드 > AI 버튼 클릭** → 플로우 AI 앱 실행
2. **플로우 채팅/게시글에서 AI 링크 클릭** → 공유 화면(웹뷰) → 원본으로 이동 또는 flow AI 클릭
3. **통합 검색 > AI 검색** → "flow AI에서 자세히 탐색하기" 클릭

### 플로우 AI → 플로우 앱 연결
- **플로우 출처 클릭** → 플로우 앱의 해당 콘텐츠로 이동

## 3. 주요 화면 상세

### 3.1 첫화면(새채팅) - FA_M_MAIN_01

#### 헤더 영역 (FA_M_MAIN_01-1)
| # | 요소 | 설명 |
|---|------|------|
| 1 | 햄버거 메뉴 | 클릭 시 사이드바 호출 |
| 2 | 플로우 AI 로고 | 클릭 시 홈(새채팅) 리프레시 |
| 3 | 프롬프트 가드 아이콘 | 보안 기능 활성 여부 표시. On: 프라이머리색+Live Dot+펄스 애니메이션 |
| 3-1 | 프롬프트 가드 아이콘 Off | 0개 활성화 상태 |

#### 메인 콘텐츠 영역 (FA_M_MAIN_01-2)
| # | 요소 | 설명 |
|---|------|------|
| 5 | 안내 문구 | "{사용자명}님, 안녕하세요. 무엇을 도와드릴까요?" |
| 6 | 퀵 액션 | 플로우 검색, 웹 검색, 이미지 생성, AI 챗봇. 토글 동작 |

#### 입력 영역
| # | 요소 | 설명 |
|---|------|------|
| 7-1 | 텍스트 입력 | 1~6줄 자동확장, 6줄 초과 시 스크롤, 플랜별 글자수 제한 |
| 7-2 | 파일 첨부 | 카메라/사진/파일/최근 업로드 파일 |
| 7-3 | 고급 답변 모드 | on/off 토글, 툴팁 5초 후 사라짐 |
| 7-4 | 기능 설정 | 자유대화(기본)/플로우검색/웹검색/이미지생성 |
| 7-5 | 검색 옵션 | 기능별 상이 (모델선택/이미지모델/프로젝트선택) |
| 7-6 | 전송 버튼 | 텍스트 미입력: 비활성 / 1글자 이상: 활성 |

### 3.2 로그인 - FA_M_MEM_01

| # | 기능 | 설명 |
|---|------|------|

## 6. 공통 인터랙션 패턴

### 텍스트 입력 (FA_M_ETC_02)
- 웹뷰에서 키패드로 인한 UI 깨짐 방지를 위해 **별도 전용 입력 화면** 사용
- 프롬프트 입력 인풋은 제외 (직접 입력)
- 취소/저장 버튼, 글자수 제한 표시

### 사용자 검색 (FA_M_ETC_01)
- 조직도 기반 뎁스 탐색
- 부서 선택 시 해당 부서 전체 선택
- 이름/부서/직급 검색 지원
