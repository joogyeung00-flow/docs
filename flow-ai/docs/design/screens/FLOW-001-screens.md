# FLOW-001 AI 스마트 비서 — 화면설계서

---

## 1. 문서 개요

본 문서는 AI 스마트 비서 "플레인"의 전체 화면 흐름과 각 화면의 UI 스펙을 정의합니다.
**Flow AI 디자인 시스템**(`design-system/design-tokens/`)의 토큰을 기준으로 설계되었습니다.

### 적용 디자인 토큰 요약

| 토큰 | 값 | 소스 |
|------|-----|------|
| 브랜드 컬러 | `brand.500` = `#5B40F8` | colors.json |
| 기본 폰트 | Pretendard | typography.json |
| 기본 단위 | 4px | spacing.json (baseUnit: 4) |
| 카드 패딩 | 24px, radius 8px | spacing.json (componentDefaults.card) |
| 채팅 버블 | px:12, py:8, radius:6 | spacing.json (componentDefaults.chatBubble) |
| 버튼 | px:16, py:8, radius:6 | spacing.json (componentDefaults.button) |
| 배지 | px:8, py:4, radius:full | spacing.json (componentDefaults.badge) |
| 애니메이션 | fadeIn 150ms, slideUp 250ms spring | motion.json |

### 화면 목록

| 화면 ID | 화면명 | 플로우 | 우선순위 |
|---------|--------|--------|---------|
| SCR-000 | 메인 채팅 화면 (Flow AI 홈) | 메인 | P0 |
| SCR-100 | AI 비서 홈 | 비서 메인 | P0 |
| SCR-101 | 브리핑 채팅 세션 | 비서 브리핑 | P0 |
| SCR-102 | 내용 브리핑 응답 | 비서 브리핑 | P0 |
| SCR-103 | 자유 대화 | 비서 대화 | P1 |
| SCR-104 | AI 비서 설정 | 비서 설정 | P1 |

## 2. 진입 구조

### 진입 경로

| 진입 경로 | 트리거 | 도착 화면 |
|----------|--------|----------|
| 사이드바 메뉴 | "AI 비서" 메뉴 탭 | SCR-100 (AI 비서 홈) |
| Push 알림 | 아침 브리핑 알림 탭 | SCR-101 (브리핑 세션 바로 진입) |

> AI 비서는 메인 홈 Quick Action에 포함되지 않음. 사이드바 전용 메뉴.

### 전체 플로우

```
[메인 채팅 화면 SCR-000]
    │
    ├── 사이드바 ☰ → "AI 비서" 탭
    │   └──▶ [AI 비서 홈 SCR-100]
    │            ├── 브리핑 카드 탭 → [브리핑 세션 SCR-101]
    │            ├── 최근 대화 탭 → 해당 세션 진입
    │            └── 입력바 → [자유 대화 SCR-103]
    │
    └── Push 알림 탭
        └──▶ [브리핑 세션 SCR-101] (바로 진입)
```

### 비서 이름/아바타 초기값

온보딩 없이 즉시 사용 가능한 구조:

| 항목 | 초기값 | 변경 방법 |
|------|--------|----------|
| 비서 이름 | "플레인" | SCR-104 설정에서 변경 |
| 아바타 | 기본 아바타 (✨) | SCR-104 설정에서 변경 |
| 말투 | 격식체 (~습니다) | SCR-104 설정에서 변경 |

**브리핑 요약 카드 스펙**

| 카드 | 아이콘 | 강조색 | 탭 동작 |
|------|--------|--------|--------|
| 일정 | 📅 | `blue.500` | 브리핑 세션의 일정 섹션으로 이동 |
| 업무 | 📋 | `orange.500` | 브리핑 세션의 업무 섹션으로 이동 |
| 변화 | 🔔 | `brand.500` | 브리핑 세션의 부재중 변화 섹션으로 이동 |

- 카드 크기: 고정 width 120px, 가로 스크롤 (snap: start)
- 건수 0인 카드: 표시하되 "없음" 텍스트, opacity 0.5

**확인 필요 카드**
- 배경: `orange.50`, border `orange.200`
- 0건일 때: 카드 숨김 처리

**개인화 넛지 배너** (비서 이름 미변경 시 최초 3회 노출)
```
┌─────────────────────────────────────┐
│ ✨ 비서의 이름을 지어주세요  [설정] │
│ (bg: brand.50, border: brand.200,   │
│  text-sm, dismissible)              │
└─────────────────────────────────────┘
```
- [설정] 탭 → SCR-104로 이동
- X 닫기 시 노출 횟수 차감, 3회 닫기 후 영구 숨김

**인터랙션**
- [전체 브리핑 보기] → SCR-101 (오늘의 브리핑 세션)
- 브리핑 카드 탭 → SCR-101 해당 섹션으로 스크롤
- 최근 대화 항목 탭 → 해당 세션 진입
- 입력바 입력 → SCR-103 (자유 대화, 새 세션 생성)
- ⚙️ 탭 → SCR-104 (설정)
- ≡ 탭 → 메인 사이드바로 복귀

**우선순위 배지 (PriorityChip 준수)**

| 레벨 | 아이콘 | 배경 | 텍스트 | 테두리 |
|------|--------|------|--------|--------|

| HIGH | 🟠 | `orange-50` | `orange-600` | `orange-200` |
| NORMAL | 🟢 | `gray.100` | `gray.600` | `gray.300` |
| LOW | ➖ | `blue-50` | `blue-600` | `blue-200` |

**CTA 버튼 ([내용 브리핑])**
- 스타일: outline, border `brand.200` (#D6D0FF), text `brand.500`
- 크기: text-xs (12px), px:8, py:4
- hover: bg `brand.50` (#F3F1FF)

**빈 상태 메시지**
- 일정: "오늘 예정된 일정이 없어요"
- 업무: "오늘과 내일 마감인 업무가 없어요"
- 부재중: "어젯밤 변화가 없어요"
- 확인필요: "모든 항목을 확인하셨어요!"

**인터랙션**

### SCR-102: 내용 브리핑 응답

**트리거**: [내용 브리핑] / [일정 브리핑] / [업무 브리핑] 버튼 클릭

**동작 흐름**
1. 버튼 비활성화 (중복 방지)
2. 타이핑 인디케이터 (● ● ● shimmer, 1초)
3. AI 답변 버블 `fade-in` (150ms)
4. 채팅 영역 자동 스크롤

**AI 답변 버블**: AI 기본 버블과 동일 스타일 (bg `gray.50`, px:12, py:8, r:6)

---

### SCR-103: 자유 대화

**동작 흐름**
1. 입력창에 텍스트 입력 → [전송] 버튼 또는 Enter
2. 사용자 버블 (우측): bg `brand.500`, text `white`, r:6
3. 타이핑 인디케이터 (1.5초)
4. AI 답변 버블 (좌측): bg `gray.50`, r:6
5. 자동 스크롤

**진입 경로별 차이**
- SCR-100 입력바에서 진입 → 새 대화 세션 생성
- SCR-101 입력바에서 진입 → 브리핑 세션 내 이어서 대화 (브리핑 컨텍스트 유지)

---

### SCR-104: AI 비서 설정

**진입**: 헤더 ⚙️ 아이콘 클릭 → 우측 슬라이드인 (250ms, ease-in-out)

**레이아웃**
```
┌─────────────────────────────────────┐
│  ← 설정  (header)                   │
├─────────────────────────────────────┤
│         (spacing-6: 24px)           │
│  프로필                             │
│  ┌─────────────────────────────┐    │
│  │  [아바타 160×160]            │    │
│  │  [아바타 변경하기] (outline)  │    │
│  └─────────────────────────────┘    │
│         (spacing-6: 24px)           │
│  비서 이름                          │
│  ┌─────────────────────────────┐    │
│  │  플레인 (input, editable)    │    │
│  └─────────────────────────────┘    │
│  0/10  (text-xs, gray.400)         │
│         (spacing-6: 24px)           │
│  말투 선택                          │
│  ○ 격식체  "~습니다, ~예요"         │
│  ○ 반말체  "~야, ~해"              │
│  ○ 귀여운체 "~용, ~해요!"          │
│  (radio, selected: brand.500)       │
│         (spacing-8: 32px)           │
│  [저장하기] (brand.500, 100%)       │
└─────────────────────────────────────┘
```

**비서 이름 유효성 검사**
- 최대 10자 (초과 입력 차단)
- 금지어: 특수문자, 예약어("Flow", "AI", "관리자")
- 빈 입력으로 [저장] 시 기본값 "플레인" 유지

**아바타 변경하기**
- [아바타 변경하기] 탭 → 키워드 선택 바텀시트
- 분위기 칩: 차분한 / 활기찬 / 따뜻한 / 시크한
- 스타일 칩: 심플 / 귀여운 / 전문적인 / 위트
- 각 카테고리 1개씩 선택 → [생성하기] → 로딩(2초) → 미리보기 → [확인] / [다시 만들기]

## 5. 네비게이션 구조

### 뒤로가기 동선

| 현재 화면 | 뒤로가기 | 방식 |
|----------|---------|------|
| SCR-000 (메인) | - | 홈이므로 뒤로가기 없음 |
| SCR-100 (AI 비서 홈) | SCR-000 사이드바 | ≡ 버튼 또는 뒤로 제스처 |
| SCR-101 (브리핑 세션) | SCR-100 | ← 버튼 |
| SCR-104 (설정) | SCR-100 | ← 버튼 |

### 전환 애니메이션

| 전환 | 애니메이션 | 토큰 |
|------|----------|------|
| 사이드바 → AI 비서 홈 | slideUp | 250ms, spring easing |
| 비서 홈 → 브리핑 세션 | slideUp | 250ms, spring easing |
| 브리핑 → 비서 홈 (뒤로) | slideDown | 250ms, spring easing |
| 비서 홈 → 설정 | slideRight | 250ms, ease-in-out |
