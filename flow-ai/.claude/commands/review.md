# 크로스 리뷰: $ARGUMENTS

## 지시사항

아래 3개의 서브에이전트를 **병렬로** 실행하여 `$ARGUMENTS`에 대해 각자 관점에서 리뷰합니다.

### 리뷰어 1: PM 관점 (@agent-pm)
- 사용자 관점에서 기능 완성도 검토
- 요구사항 누락 여부 확인
- 엣지 케이스 및 예외 시나리오 점검
- 비즈니스 임팩트 평가

### 리뷰어 2: Designer 관점 (@agent-designer)
- UX 일관성 및 사용성 검토
- 디자인 시스템 준수 여부 확인
- 접근성(A11y) 이슈 점검
- 인터랙션 완성도 평가

### 리뷰어 3: Developer 관점 (@agent-dev)
- 기술적 실현 가능성 검토
- 성능 및 확장성 이슈 점검
- 코드 품질 및 유지보수성 평가
- 보안 관련 우려사항 확인

### 결과 취합
모든 리뷰가 완료되면 결과를 취합하여 정리합니다:
- **Critical**: 반드시 수정 필요
- **Major**: 수정 권장
- **Minor**: 개선 제안
- **Good**: 잘된 점
